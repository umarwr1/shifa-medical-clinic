# Shifa Medical Clinic — Website (previous-style replica)

A self-contained static recreation of the previous Shifa Medical Clinic website,
based on the reference screenshots. This folder is completely separate from the
existing project and does not affect it.

## How to view

Just double-click `index.html` to open it in any web browser. No installation or
build step is required. (An internet connection is recommended so the web fonts,
icons, map and stock photos load.)

## Pages

- `index.html` — Home (hero slider, feature cards, why-choose-us + feedback, services, doctors)
- `about.html` — About Us, Our Mission, Why Choose Us, Our Team
- `services.html` — All services with detailed descriptions
- `blogs.html` — Blog/article grid
- `gallery.html` — Photo gallery
- `contact.html` — Map, contact form and clinic details

## Folder structure

```
Shifa-Medical-Clinic-Site/
├── index.html / about.html / services.html / blogs.html / gallery.html / contact.html
├── css/style.css
├── js/main.js
├── images/         (clinic photos)
└── README.md
```

## Replacing placeholder photos

The hero slider, blog covers and gallery currently use stock photos and the
Google Maps embed. To use your own pictures:

1. Put your images in the `images/` folder.
2. In the relevant HTML file, change the image URL to e.g. `images/your-photo.jpg`.
